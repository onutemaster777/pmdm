sgtelement.js - animated function style innovation tool (AFSIT)

sgtelement.js - the animated function style innovation tool that it animates emotion animations, jumping, shaking, bouncing, click to edit, rotation, and more. This script will let you to apply the element with style.

A individual function is to apply the function from named element.

Here's some functions that you can apply the element

  • sgt - when executed, the warning will appear from the console (press F12 or right-click anywhere in this page to open the developer tools from context menu)
  • sgtRotate - Spin around. (refer as "sgt.prototype.rotate")
  • sgtShake - Shake the element, trembling or earthquake element. (refer as "sgt.prototype.shake")
  • sgtCameraShot - Simulute a camera shot. (refer as "sgt.prototype.cameraShot")
  • sgtZap - Applies Zap function "struck the element". It will change into yellow, blue, or cyan for every millisecond. (refer as "sgt.prototype.zap")
  • sgtLoadingMouse - Simulate mouse waiting when doing the process (needs restore with another). (refer as "sgt.prototype.waitingMouse")
  • sgtRestoreMouse - Restore simulated mouse. (refer as "sgt.prototype.waitingMouseRestore")
  • sgtDisableElement - Disable a element (back it) (refer as "sgt.prototype.disable")
  • sgtEnableElement - Enable a element (restore) (refer as "sgt.prototype.enable")
  • sgtPopup - Simuluate a popup animation. (refer as "sgt.prototype.popup")
  • sgtDisableGray - Disable element with grayscale effect. (refer as "sgt.prototype.disableGray")
  • sgtBlurInappropriate - Blur the element contains inappropriate content. (refer as "sgt.prototype.blurInappropriate")
  • sgtWarnNSFW - Blur the element, clicking the element will warn the client to remove the blur. (refer as "sgt.prototype.warnInappropriate")
  • sgtBanNSFW - Blur the element, clicking the element will warn the client and it will not remove the blur and replace element to banned text. (refer as "sgt.prototype.banInappropriate")
  • sgtPop - Remove before popping. (refer as "sgt.prototype.pop")
  • sgtSquash - Squash the element. (refer as "sgt.prototype.squash") (uses two functions)
  • sgtSqueeze - Squeeze the element. (refer as "sgt.prototype.squeeze") (uses two functions) (e.g. sgt.prototype.squeeze("test", 0.5)
  • sgtFlop - Flip the element. (refer as "sgt.prototype.flop")
  • sgtWaitForget - Focus/scrolls down and flashes element. (refer as "sgt.prototype.waitForget")
  • sgtError - Turn border to red. (refer as "sgt.prototype.error")
  • sgtSuccess - Turn border to lime. (refer as "sgt.prototype.success")
  • sgtOkay - Turn border to orange. (refer as "sgt.prototype.okay")
  • sgtInfo - Turn border to blue. (refer as "sgt.prototype.info")
  • sgtNeutral - Turn border to normal. (refer as "sgt.prototype.normal")
  • sgtClickSim - Highlight and clicks the element. (refer as "sgt.prototype.clickSim")
  • sgtMagicDelete - Deletes and fades the element (refer as "sgt.prototype.magicDelete")
  • sgtFullscreen - Enter element to fullscreen mode. (refer as "sgt.prototype.fullscreen")
