What is Internet Time Travel Browser

Internet Time Travel Browser (shortly known as "ITT" or "ITTB") is a website that travels back to 2004 pages by using Wayback Machine.

Internet Time Travel Browser has a minimal features which you have a new browser, but you want old web? It supports old computers to travel across the web via Wi-Fi alliance connection. Other websites that you're trying to connect are different dates, because of Internet Archive didn't index it recursively.

As you expect, you also just mess it around with this website. This source is free, feel to download the page and put your own page, visitors will enjoy your old web by using Wayback Machine.
